# Student Registration Multi-Step Form

## Overview
This project is a **multi-step student registration form** built with **HTML, CSS, and JavaScript**.  
It allows students to input personal, academic, and skills information step by step, review their entries, and submit the form. After submission, a success message is displayed. Users can also go back to edit their information before submitting.

## Features
- Multi-step form with a sidebar progress indicator.
- Review page to double-check submitted information.
- **Edit button** to go back and modify information.
- Mobile responsive design.
- Displays a success message after submission.

## Validation
- Email field is validated using a regular expression.
- Prevents navigation to the next step if the email format is invalid.
- Displays an error message for incorrect email input.

## How to Use
1. Open `index.html`in your browser or click the live demo link.
2. Fill in the **Personal Information**, **Academic Information**, and **Skills & Competencies** steps.
3. Click **Next Step** to navigate through the form.
4. On the **Review & Submit** page, you can:
   - Click **Edit** to go back and modify any information.
   - Click **Submit Registration** to complete the process and see the success message.
